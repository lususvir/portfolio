Interaction Design

ALEKS Input Tools

Web Design

Background

Due to updates in common core math standards, ALEKS was tasked with introducing new items into its college algebra and geometry curriculum. Such items required students to move, rotate, translate, and reflect objects over X and Y axes.

Prior to my arrival, UX designers had sketched concepts but ultimately relied on developers to create the tool in order to test it with real users. Due to the high investment of developing the tools, often times problems uncovered in user testing were not implemented for resourcing reasons.

Process

I was able to use Axure to prototype tools that closely mimicked how a real tool would behave. This allowed us to conduct usability testing without any actual development work.

There were many design constraints. ALEKS is a robust math learning platform with hundreds of mathematical items already built. Because of that, and due to some technological constraints, much of the design of the tool was dictated by existing tools, such as:       

The size of the graphing area       The size, look, and feel of the tool buttons The pencil, eraser, plot a point and "graph" functions are part of the standard library The "delete", "undo", and "help" buttons (dark green) are existing functions

Findings

We conducted in-house usability testing with kids from local schools. We tested each of these items by asking the student to read a directions and then use the tool to accomplish the task. Because we were able to test the tools with real participants, we uncovered a few problems with the original designs. 

For example, the polynomial graphing tool was originally designed so participants selected the intercept behavior before placing it on the graph (the tool box on the right had each individual behavior as a separate button). Participants failed to place the correct behavior due to lack of context.

We changed the design to allow students to place a generic point on the graph (they're used to doing that in other items), and then select the behavior once it was placed. This solution tested very well and students seemed to understand how to complete the item.
